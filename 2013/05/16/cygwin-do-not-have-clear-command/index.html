Cygwin 默认没有 clear 清屏命令!

By Ray Lee

5月 16 2013 更新日期:4月 26 2014

文章目录

我使用 Linux 清屏的习惯是敲clear命令, 今天在 Cygwin 中使用clear命令却出现-bash: clear: command not found, 于是找到了几个清屏的方法:

  1. bash script处理 : echo -e "\E[2J"^C
  2. 使用快捷键 : Ctrl + L
  3. 运行 Cygwin Setup 安装 ncurses (in Utils) 包, 就可以使用 clear 命令了
